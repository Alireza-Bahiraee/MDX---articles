# MDX Query Notes & Articles

A personal repository for Markdown notes, examples, and articles about MDX (Multidimensional Expressions) used with SQL Server Analysis Services (SSAS) and other OLAP engines.

## Purpose

This repo is my working knowledge base for:

- MDX syntax and patterns
- Common query templates (filters, rankings, time intelligence, etc.)
- Debugging and performance tips
- Concepts around multidimensional models, slices, sets, and measures
- Connections between MDX, DAX, and semantic modeling

It’s meant to be practical, opinionated, and evolving as I learn more.


## How to Use

Most files are plain Markdown and can be:

- Read in GitHub/GitLab
- Opened locally with any Markdown editor
- Converted to PDF/HTML if you want offline notes
